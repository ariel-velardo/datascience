Minha pesquisa investiga possibilidades de aplicação de técnicas de aprendizado de máquina quântico em problemas de alta complexidade, com interesse especial em aplicações no mercado de crédito, modelagem preditiva, explicabilidade e sistemas analíticos orientados à decisão.

Este repositório reúne meus estudos, projetos práticos e experimentações em Ciência de Dados. A ideia é manter um espaço organizado para praticar, testar abordagens, documentar aprendizados e evoluir tecnicamente.
