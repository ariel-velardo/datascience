# 05 - Recomendação Executiva

Tarefa:
1. Resumir principais achados.
2. Separar fato, hipótese e recomendação.
3. Explicar impacto esperado.
4. Apontar riscos e limitações.
5. Sugerir próxima ação.
6. Traduzir resultado técnico em decisão de negócio.

Formato:
- Diagnóstico
- Evidências
- Oportunidade
- Recomendação
- Próximos passos

Restrições:
- Ser direto.
- Não exagerar causalidade.
- Usar linguagem executiva.
