# 02 - Definição Causal

Tarefa:
1. Definir unidade de análise.
2. Definir tratamento.
3. Definir controle.
4. Definir outcome.
5. Definir janela pré e pós.
6. Listar covariáveis permitidas.
7. Apontar riscos de leakage.
8. Dizer se a análise permite inferência causal ou apenas associação.

Restrições:
- Usar somente variáveis pré-tratamento como covariáveis.
- Não assumir causalidade sem hipótese clara.
- Separar análise descritiva, preditiva e causal.
